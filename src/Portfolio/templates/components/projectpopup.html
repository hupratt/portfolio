Coach is a calendar app built with Django on the backend and react in the frontend. The app is bundled with Webpack and transpiled with Babel. The app features OAuth authentication with facebook google and github. The frontend is integrated into django as an "app". Python web application

Ecommerce

Ecom is a django single page (SPA) web app running on apache as a web server.
The latest multiprocessing apache module is used to create parallel replicas of our app. Each apache "event_worker" is 1 process and 1 python app. 1 process launches multiple threads so that each app is concurrently accessed by multiple users. This allows us to have replication without installing Docker
